发展历程
------
### 刚开始时
本意是写一些初学C常写的代码示例，但发现自己写不来C。在初学程序语言的时候就没分清C/C++，自己基础也不好。自己一懒，就开始想C/C++都收录。

### 回归 C语言
开始学习《C语言编程魔法书》，正式的从基础开始学习C语言。在学习的过程中重新开始写这个项目吧。

### 开始每周一码
发现自己学习计划不能稳定进行，开启这个低难度的计划。
